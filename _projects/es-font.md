In the Endless Universe, most factions, but especially anything built by the Endless themselves, use the Endless Script. While the letters look different, the text itself is plain English and, in some cases, (fake) Latin.
The font that you can download in the top right of the page is my own recreation of it. Some years ago and after a scavenger hunt, Amplitude actually published the cypher to the script in a Twitter post, that unfortunately is not online anymore. At some later point, the cypher was also [posted by Sublustris](https://community.amplitude-studios.com/amplitude-studios/endless-space-2/forums/65-general/threads/24939-endless-alphabet#post-232595) on an Amplitude fan forum.
Would I have known about this post at the time, this would have been a huge time saver but alas, I did everything myself.


### The font

What the font does on purpose not have are any kind of punctuation marks. This is because I could only find one instance, where a '-' is used. For now dots, commas, exclamtion points, and question marks are replaced by a whitespace.

Note that it seems that the font used by Amplitude internally for in-game artworks and the UI seems to be a monospaced font. Since I used this project to learn how to create a font, I added kerning, which as a first try turned out pretty ok. <span class='es-font-dynamic'>I hope you agree on this. Almost looks like a real font, no?</span>
